# Grid

Smooth UI has the most powerful grid system to layout and align content. It's built with [flexbox](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Flexible_Box_Layout/Using_CSS_flexible_boxes), mobile first and fully responsive.

It implements all feature of great [Bootstrap grid](https://getbootstrap.com/docs/4.1/layout/grid/) with the power of React.

<Playground>
  <Grid>
    <Row>
      <Col>One</Col>
      <Col>Two</Col>
      <Col>Three</Col>
    </Row>
  </Grid>
</Playground>

The above example creates three equal-width columns on small, medium, large, and extra large devices.

## Grid component

`Grid` component is a layout element, it is required to use grid system.  Choose from a responsive, fixed-width container (meaning its `max-width` changes at each breakpoint) or fluid-width (meaning it’s `100%` wide all the time).

<Playground>
  <Grid>
    This is a fixed grid
  </Grid>
</Playground>

Use `fluid` prop for a full width container, spanning the entire width of the viewport.

<Playground>
  <Grid fluid>
    This is a fluid grid
  </Grid>
</Playground>

## Auto-layout columns

### Equal-width

For example, here are two grid layouts that apply to every device and viewport, from `xs` to `xl`. Add any number of unit-less props for each breakpoint you need and every column will be the same width.

<Playground>
  <Grid>
    <Row>
      <Col>1 of 2</Col>
      <Col>2 of 2</Col>
    </Row>
    <Row>
      <Col>1 of 3</Col>
      <Col>2 of 3</Col>
      <Col>3 of 3</Col>
    </Row>
  </Grid>
</Playground>

### Equal-width multi row

Create equal-width columns that span multiple rows by inserting a `Col.Split` where you want the columns to break to a new line.

<Playground>
  <Grid>
    <Row>
      <Col>Column</Col>
      <Col>Column</Col>
      <Col.Split />
      <Col>Column</Col>
      <Col>Column</Col>
    </Row>
  </Grid>
</Playground>

### Setting one column width

Auto-layout for flexbox grid columns also means you can set the width of one column and have the sibling columns automatically resize around it.

<Playground>
  <Grid>
    <Row>
      <Col>1 of 3</Col>
      <Col xs={6}>2 of 3 (wider)</Col>
      <Col>3 of 3</Col>
    </Row>
    <Row>
      <Col>1 of 3</Col>
      <Col xs={5}>2 of 3 (wider)</Col>
      <Col>3 of 3</Col>
    </Row>
  </Grid>
</Playground>

### Variable width content

Use "auto" to size columns based on the natural width of their content.

<Playground>
  <Grid>
    <Row justifyContent={{ md: 'center' }}>
      <Col lg={2}>1 of 3</Col>
      <Col xs="auto">Variable width content</Col>
      <Col lg={2}>3 of 3</Col>
    </Row>
    <Row>
      <Col>1 of 3</Col>
      <Col xs="auto">Variable width content</Col>
      <Col>3 of 3</Col>
    </Row>
  </Grid>
</Playground>

## Responsive classes

Smooth UI's grid includes five tiers of predefined classes for building complex responsive layouts. Customize the size of your columns on extra small, small, medium, large, or extra large devices however you see fit.

### All breakpoints

For grids that are the same from the smallest of devices to the largest, use the props `xs`, `sm`, `md`, `lg` or `xl`. Specify a number value when you need a particularly sized column; otherwise, feel free to put the prop without a value (`true`).

<Playground>
  <Grid>
    <Row>
      <Col>-</Col>
      <Col>-</Col>
      <Col>-</Col>
      <Col>-</Col>
    </Row>
    <Row>
      <Col xs={8}>{`xs={8}`}</Col>
      <Col xs={4}>{`xs={4}`}</Col>
    </Row>
  </Grid>
</Playground>

### Stacked to horizontal

Using a single set of `sm` props, you can create a basic grid system that starts out stacked before becoming horizontal with at the small breakpoint (`sm`).

<Playground>
  <Grid>
    <Row>
      <Col sm={8}>{`sm={8}`}</Col>
      <Col sm={4}>{`sm={4}`}</Col>
    </Row>
    <Row>
      <Col>col</Col>
      <Col>col</Col>
      <Col>col</Col>
    </Row>
  </Grid>
</Playground>

### Mix and match

Don’t want your columns to simply stack in some grid tiers? Use a combination of different props for each tier as needed. See the example below for a better idea of how it all works.

<Playground>
  <Grid>
    {/* Stack the columns on mobile by making one full-width and the other half-width */}
    <Row>
      <Col xs={12} md={8}>{`xs={12} md={8}`}</Col>
      <Col xs={6} md={4}>{`xs={6} md={4}`}</Col>
    </Row>
    {/* Columns start at 50% wide on mobile and bump up to 33.3% wide on desktop */}
    <Row>
      <Col xs={6} md={4}>
        {`xs={6} md={4}`}
      </Col>
      <Col xs={6} md={4}>
        {`xs={6} md={4}`}
      </Col>
      <Col xs={6} md={4}>
        {`xs={6} md={4}`}
      </Col>
    </Row>
    {/* Columns are always 50% wide, on mobile and desktop */}
    <Row>
      <Col xs={6}>
        {`xs={6}`}
      </Col>
      <Col xs={6}>
        {`xs={6}`}
      </Col>
    </Row>
  </Grid>
</Playground>

## Custom gutter

To change default custom gutter, use `gridGutter` in theme.

You can also specify a custom gutter using `gutter` property on `Grid`, `Row` and `Col`.

<Playground>
  <Grid gutter={5}>
    <Row gutter={5}>
      <Col gutter={5}>{`5px gutter`}</Col>
      <Col gutter={5}>{`5px gutter`}</Col>
      <Col gutter={5}>{`5px gutter`}</Col>
    </Row>
  </Grid>
</Playground>

### No gutter

To remove all gutter, set them to `0`.

<Playground>
  <Grid gutter={0}>
    <Row gutter={0}>
      <Col gutter={0}>{`no gutter`}</Col>
      <Col gutter={0}>{`no gutter`}</Col>
      <Col gutter={0}>{`no gutter`}</Col>
    </Row>
  </Grid>
</Playground>

### No external gutter

To avoid external gutter, just remove the `Grid` or set `gutter` to `0` only in grid.

<Playground>
  <Grid gutter={0}>
    <Row>
      <Col>{`No external gutter`}</Col>
      <Col>{`No external gutter`}</Col>
      <Col>{`No external gutter`}</Col>
    </Row>
  </Grid>
  <Grid>
    <Row>
      <Col>{`External gutter`}</Col>
      <Col>{`External gutter`}</Col>
      <Col>{`External gutter`}</Col>
    </Row>
  </Grid>
</Playground>

## Column wrapping

If more than 12 columns are placed within a single row, each group of extra columns will, as one unit, wrap onto a new line.

<Playground>
  <Row>
    <Col xs={9}>{`xs={9}`}</Col>
    <Col xs={4}>{`xs={4}`}<br />Since 9 + 4 = 13 &gt; 12, this 4-column-wide gets wrapped onto a new line as one contiguous unit.</Col>
    <Col xs={6}>{`xs={6}`}<br />Subsequent columns continue along the new line.</Col>
  </Row>
</Playground>

## Column breaks

Breaking columns to a new line in flexbox requires a small hack: add an element with `width: 100%` wherever you want to wrap your columns to a new line. Normally this is accomplished with multiple `Row`, but not every implementation method can account for this.

<Playground>
  <Row>
    <Col xs={6} sm={3}>{`xs={6} sm={3}`}</Col>
    <Col xs={6} sm={3}>{`xs={6} sm={3}`}</Col>

    {/* Force next columns to break to new line */}
    <Col.Split></Col.Split>

    <Col xs={6} sm={3}>{`xs={6} sm={3}`}</Col>
    <Col xs={6} sm={3}>{`xs={6} sm={3}`}</Col>
  </Row>
</Playground>

You may also apply this break at specific breakpoints with our responsive display utilities.

<Playground>
  <Row>
    <Col xs={6} sm={4}>{`xs={6} sm={4}`}</Col>
    <Col xs={6} sm={4}>{`xs={6} sm={4}`}</Col>

    {/* Force next columns to break to new line at md breakpoint and up */}
    <Col.Split display={{ xs: 'none', md: 'block' }}></Col.Split>

    <Col xs={6} sm={4}>{`xs={6} sm={4}`}</Col>
    <Col xs={6} sm={4}>{`xs={6} sm={4}`}</Col>
  </Row>
</Playground>

## Reordering

### Order prop

Use `order` prop for controlling the visual order of your content. Like all utilities, it supports responsive, you can use `order={{ sm: 2 }}`.

<Playground>
  <Grid>
    <Row>
      <Col>
        First, but unordered
      </Col>
      <Col order={2}>
        Second, but last
      </Col>
      <Col order={1}>
        Third, but first
      </Col>
    </Row>
  </Grid>
</Playground>

### Offsetting columns

You can offset grid columns in two ways: our responsive offset props and our margin utilities. Grid classes are sized to match columns while margins are more useful for quick layouts where the width of the offset is variable.

#### Offset props

Move columns to the right using offset props (`oxs`, `osm`...). These props increase the left margin of a column by * columns. For example, `omd={4}` moves `Col` over four columns.

<Playground>
  <Row>
    <Col md={4}>{`md={4}`}</Col>
    <Col md={4} omd={4}>{`md={4} omd={4}`}</Col>
  </Row>
  <Row>
    <Col md={3} omd={3}>{`md={3} omd={3}`}</Col>
    <Col md={3} omd={3}>{`md={3} omd={3}`}</Col>
  </Row>
  <Row>
    <Col md={6} omd={3}>.{`md={6} omd={3}`}</Col>
  </Row>
</Playground>

In addition to column clearing at responsive breakpoints, you may need to reset offsets.

<Playground>
  <Row>
    <Col sm={5} md={6}>{`sm={5} md={6}`}</Col>
    <Col sm={5} osm={2} md={6} omd={0}>{`sm={5} osm={2} md={6} omd={0}`}</Col>
  </Row>

  <Row>
    <Col sm={6} md={5} lg={6}>{`sm={6} md={5} lg={6}`}</Col>
    <Col sm={6} md={5} omd={2} lg={6} olg={0}>{`sm={6} md={5} omd={2} lg={6} olg={0}`}</Col>
  </Row>
</Playground>

#### Margin utilities

You can use margin utilities to force sibling columns away from one another.

<Playground>
  <Row>
    <Col xs={4}>{`xs={4}`}</Col>
    <Col xs={4} ml="auto">{`xs={4} ml="auto"`}</Col>
  </Row>
  <Row>
    <Col xs={3} ml={{ md: 'auto' }}>{`xs={3} ml={{ md: 'auto' }}`}</Col>
    <Col xs={3} ml={{ md: 'auto' }}>{`xs={3} ml={{ md: 'auto' }}`}</Col>
  </Row>
  <Row>
    <Col xs="auto" mr="auto">{`xs="auto" mr="auto"`}</Col>
    <Col xs="auto">{`xs="auto"`}</Col>
  </Row>
</Playground>

## Nesting

To nest your content with the default grid, add a new `Row` and set of `<Col sm={*} />` within an existing `<Col sm={*} />`. Nested rows should include a set of columns that add up to 12 or fewer (it is not required that you use all 12 available columns).

<Playground>
  <Row>
    <Col sm={9}>
      {`Level 1: sm={9}`}
      <Row>
        <Col xs={8} sm={6}>
          {`Level 2: xs={8} sm={6}`}
        </Col>
        <Col xs={4} sm={6}>
          {`Level 2: xs={4} sm={6}`}
        </Col>
      </Row>
    </Col>
  </Row>
</Playground>

## Customize

Grid defaults can be customized in theme, these are defaults:

```js
export const gridColumns = 12
export const gridGutter = 16

export const breakpoints = {
  // Extra small screen / phone
  xs: 0,
  // Small screen / phone
  sm: '576px',
  // Medium screen / tablet
  md: '768px',
  // Large screen / desktop
  lg: '992px',
  // Extra large screen / wide desktop
  xl: '1200px'
}

export const gridMaxWidths = {
  sm: '540px',
  md: '720px',
  lg: '960px',
  xl: '1140px'
}
```
