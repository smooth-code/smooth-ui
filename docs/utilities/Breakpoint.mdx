# Breakpoint

Breakpoint is an utility to show or hide a block relatively to a breakpoint. 

<Playground>
  <Breakpoint down="md">
    <Button variant="primary">Button displayed down to md</Button>
  </Breakpoint>
  <Breakpoint up="md">
    <Button variant="secondary">Button displayed up to md</Button>
  </Breakpoint>
</Playground>
