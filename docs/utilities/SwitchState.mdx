# SwitchState

SwitchState is a render-props component that provide all internal thing to build a `radio` or `checkbox` based button. `Checkbox`, `Radio` and `Switch` component are using it. It works for controlled and uncontrolled components.

## Create a custom checkbox / radio

`SwitchState` exposes the following render props:

- `checked`: is the input checked?
- `focused`: is the input focused?
- `disabled`: is the input disabled?
- `input`: contains all necessary input props

To create a checkbox (or a radio), two things are required:

- Add a `<input />` and spread `input`
- Add a parent container with `position: relative`

You decide other things! In this example, I used a customized `Box` to create it.

For more inspiration, take a look in the source code of the project, the `Switch` is a good customization example.
