# Toggler

Toggler is a render-props component that provide a toggle behaviour.

<Playground>
  <Toggler>
    {({ toggled, onToggle }) => (
      <Button onClick={onToggle}>{toggled ? 'ON' : 'OFF'}</Button>
    )}
  </Toggler>
</Playground>

## Specify a value in `onToggle`

<Playground>
  <Toggler>
    {({ toggled, onToggle }) => (
      <div className="block-list">
        <div>{toggled ? 'ðŸŒž' : 'ðŸŒ›'}</div>
        <div>
          <Button size="sm" onClick={() => onToggle(false)}>
            Night
          </Button>{' '}
          <Button size="sm" onClick={() => onToggle(true)}>
            Day
          </Button>
        </div>
      </div>
    )}
  </Toggler>
</Playground>

## Set the default toggle

You can define the default `toggled` value using `defaultToggled` property.

<Playground>
  <Toggler defaultToggled>
    {({ toggled, onToggle }) => (
      <Button onClick={onToggle}>{toggled ? 'ON' : 'OFF'}</Button>
    )}
  </Toggler>
</Playground>

## Listen for toggling

You can listen when the element is toggled using `onToggle` property.

<Playground>
  <Toggler onToggle={toggled => console.log('Toggled', toggled)}>
    {({ toggled, onToggle }) => (
      <Button onClick={onToggle}>{toggled ? 'ON' : 'OFF'}</Button>
    )}
  </Toggler>
</Playground>
