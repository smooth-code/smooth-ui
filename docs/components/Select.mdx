# Select

```js
import { Select } from '@smooth-ui/core-sc'
```

## Sizes

Set sizes using `size` prop like `"sm"`, `"md"` or `"lg"`.

<Playground>
  <BlockList>
    <Select size="sm" placeholder="Small">
      <option value="one">One</option>
      <option value="two">Two</option>
    </Select>
    <Select size="md" placeholder="Medium">
      <option value="yes">Yes</option>
      <option value="no">No</option>
      <option value="maybe">Maybe</option>
    </Select>
    <Select size="lg" placeholder="Large">
      <option value="amazing">Amazing</option>
      <option value="great">Great</option>
    </Select>
  </BlockList>
</Playground>

## Disabled

Disable using `disabled` prop.

<Playground>
  <Select disabled placeholder="Disabled">
    <option>Disabled</option>
  </Select>
</Playground>

## Control & Validation

* Set `display` to "block" and `width` to "100%" using `control` prop.
* Set validation using `valid` or `valid={false}`

<Playground>
  <BlockList>
    <Select control>
      <option>Control</option>
      <option>Other</option>
    </Select>
    <Select control valid>
      <option>Valid</option>
      <option>Other</option>
    </Select>
    <Select control valid={false}>
      <option>Invalid</option>
      <option>Other</option>
    </Select>
  </BlockList>
</Playground>
