# Checkbox

## Sizes

Set sizes using `size` prop like `"sm"`, `"md"` or `"lg"`.

<Playground>
  <FormCheck>
    <Checkbox
      size="sm"
      id="sizeCheckbox1"
      name="sizeCheckboxes"
      value="option1"
    />
    <FormCheckLabel htmlFor="sizeCheckbox1">Small</FormCheckLabel>
  </FormCheck>
  <FormCheck>
    <Checkbox id="sizeCheckbox2" name="sizeCheckboxes" value="option2" />
    <FormCheckLabel htmlFor="sizeCheckbox2">Medium</FormCheckLabel>
  </FormCheck>
  <FormCheck>
    <Checkbox
      size="lg"
      id="sizeCheckbox3"
      name="sizeCheckboxes"
      value="option3"
    />
    <FormCheckLabel htmlFor="sizeCheckbox3">Large</FormCheckLabel>
  </FormCheck>
</Playground>

## Disabled

Disable using `disabled` prop.

<Playground>
  <FormCheck>
    <Checkbox
      disabled
      id="disabledCheckbox1"
      name="disabledCheckboxs"
      value="option1"
    />
    <FormCheckLabel htmlFor="disabledCheckbox1">Disabled</FormCheckLabel>
  </FormCheck>
</Playground>

## Control & Validation

- Set custom focus effect using `control` prop.
- Set validation using `valid` or `valid={false}`

<Playground>
  <FormCheck>
    <Checkbox
      id="controlCheckbox1"
      name="controlCheckboxes"
      value="option1"
      control
    />
    <FormCheckLabel htmlFor="controlCheckbox1">Control</FormCheckLabel>
  </FormCheck>
  <FormCheck>
    <Checkbox
      id="controlCheckbox2"
      name="controlCheckboxes"
      value="option2"
      control
      valid
    />
    <FormCheckLabel htmlFor="controlCheckbox2">Valid Control</FormCheckLabel>
  </FormCheck>
  <FormCheck>
    <Checkbox
      id="controlCheckbox3"
      name="controlCheckboxes"
      value="option3"
      control
      valid={false}
    />
    <FormCheckLabel htmlFor="controlCheckbox3">Invalid Control</FormCheckLabel>
  </FormCheck>
</Playground>
